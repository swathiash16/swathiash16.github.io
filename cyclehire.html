Analysis of Cycle Hire Dataset using PySpark

Leveraged PySpark for scalable data analysis of a cycle hire dataset, encompassing data preprocessing, exploratory data analysis (EDA), and feature engineering to optimize dataset suitability for analysis. Applied advanced statistical techniques, including the t-test, to investigate and compare ride duration metrics, providing actionable insights into usage patterns and operational efficiencies. Communicated findings effectively through compelling visualizations using PySpark and Matplotlib, facilitating data-driven decision-making and operational improvements.

Detailed results and methodology can be found in the project notebook and the project report.
