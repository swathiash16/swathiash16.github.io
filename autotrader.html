Prediction of Autotrader Car Prices

Developed a predictive model to estimate car prices using historical data from Autotrader. Utilized Python and Scikit-learn for data pre-processing, feature engineering, and model building. Implemented various regression algorithms such as Linear Regression, Decision Trees, Ensemble Model, and Random Forests. Evaluated model performance using metrics like RMSE, MAE, and R-squared. Improved model accuracy by 15% through hyperparameter tuning and cross-validation. Visualized the results using Matplotlib and Seaborn to provide insights into the factors affecting car prices.

Detailed results and methodology can be found in the project notebook and the project report.
